# Contributing

Contributions are welcome when they preserve the project's independent visual identity and licensing clarity.

## Before submitting

1. Do not include screenshots, characters, layouts, prompts, or example images copied from third parties.
2. Do not imitate a living artist or identifiable character.
3. Keep 小K's fixed character DNA unchanged unless the proposal explicitly updates the character specification.
4. Keep each illustration focused on one action, judgment, or state.
5. Run the repository checks before opening a pull request.

```powershell
python .\scripts\validate_package.py .
python "$env:USERPROFILE\.codex\skills\.system\skill-creator\scripts\quick_validate.py" .\xiao-k-illustration
```

## Asset contributions

For every new visual asset, include:

- source format when available
- author or generator information
- prompt and meaningful human edits for AI-assisted work
- license confirmation

By contributing, you agree that code and Skill instructions are licensed under MIT and original visual assets are licensed under CC BY 4.0.
