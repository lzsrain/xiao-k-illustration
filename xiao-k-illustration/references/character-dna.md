# 小K Character DNA

## Identity

小K是一只总在赶时间、略显疲惫但仍继续做事的小牛。它代表使用 AI 解决真实问题的普通行动者，不是专家、老板、超级英雄或卖萌吉祥物。

## Six fixed anchors

1. 圆润、宽厚、略向前倾的小牛身体，头身连成一个整体，轮廓接近柔和的圆角梯形。
2. 面部中央是一条白色横向眼窗，内有两个下垂黑色小眼，默认困倦但仍清醒。
3. 头顶有两只短小弯角，左右基本平衡但保留手绘差异；不添加动物耳朵。
4. 戴醒目的暖黄色包耳耳机，黑色头梁从两角之间经过，耳罩贴合身体两侧。
5. 尾巴末端是一只小钟表，表针可以参与情绪和叙事。
6. 两条弹簧腿承受笨重身体，脚穿实心黑色短靴。

## Face

- 小牛口鼻直接融入身体前缘，不使用独立方块鼻子。
- 口鼻只保留两个小鼻孔和一条短嘴线，不能画成猪鼻或写实牛脸。
- 情绪优先通过眼窗内的眼睛、身体倾角、弹簧腿和尾钟表现。

## Drawing language

- 使用粗糙黑色蜡笔、铅笔或干墨线，允许轻微抖动和不均匀笔触。
- 身体填充暖白或极浅冷灰；轮廓、手臂、尾巴和腿为炭黑。
- 黄色只用于耳机、对勾、关键模块等少量强调物。
- 背景默认纯白或暖白，不使用渐变、写实光影和 3D 渲染。

## Limbs and props

- 手臂是黑色火柴线，手为圆点或极简手套，可按动作伸缩。
- 弹簧腿可以压缩、拉长和打结，但靴子保持黑色短靴。
- 钟表尾巴必须保留，表针可表示赶时间、停滞或完成。
- 公文包、工牌、便签、电脑和手机都是场景道具，不属于角色本体。

## Expression system

- 兴奋：眼睛弯起，双手抬高，弹簧腿弹起，尾钟向前。
- 疑惑：身体倾斜，一手碰耳机，双眼错向，尾钟向后。
- 崩溃：眼睛下垂，身体塌下，弹簧腿打结，尾钟快速旋转。
- 搞定：眼睛弯起，一手点赞，腿恢复稳定，尾钟指向完成。

## Never change

- 圆润笨重的小牛轮廓
- 白色横向困眼窗
- 两只短弯角且没有耳朵
- 大黄色耳机
- 钟表尾巴
- 弹簧腿与黑色短靴

## Never add by default

- 胸口字母或 Logo
- 独立方块鼻子、写实牛鼻或猪鼻
- 动物耳朵、翅膀、鸟喙
- 服装、围巾或帽子
- 毛线球、土豆或黑色实心身体
- 复杂办公室背景
